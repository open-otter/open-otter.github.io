# OpenOtter Docs

OpenOtter is an open-source platform for building and operating autonomous RC cars with a separate vehicle brain, motor controller, and RC chassis. These docs are the lightweight site entry point: enough structure to get a build started, with room for maintainers to fill in project-specific details as the hardware and software stack evolves.

## Starter guides

- [Hardware Build](/docs/hardware-build): chassis selection, wiring, power layout, and sensor mounting
- [Software Build](/docs/software-build): iOS app setup, STM32 firmware toolchain, and local build workflows
- [Running Instructions](/docs/running-instructions): bring-up order, connection checks, and safe operating flow
